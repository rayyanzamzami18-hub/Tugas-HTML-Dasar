Projek SKL Web Perpustakaan

Project SKL ini adalah pembuatan website perpustakaan berbasis web yang menggunakan database PostgreSQL untuk menyimpan dan mengelola data. Website ini dibuat untuk memudahkan pengelolaan data buku, data anggota, serta proses peminjaman dan pengembalian buku agar lebih teratur dan efisien. Website ini dikembangkan menggunakan Python sebagai backend dan PostgreSQL sebagai sistem databasenya. Setelah selesai dibuat, aplikasi ini di-online-kan menggunakan PythonAnywhere sehingga bisa diakses melalui internet. Melalui project ini, saya belajar menerapkan konsep database, pengelolaan data, serta proses deploy aplikasi web ke server secara langsung.
